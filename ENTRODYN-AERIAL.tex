    {\centering \textbf{\textcolor{ChemathonBlue}{\normalsize Proposed Solution}}\par}
    
    \vspace{0.1cm}
    \begin{itemize} \itemsep0.1em
        \item \textbf{Proposed Solution:} \\
        A cyber-physical drone ecosystem utilizing \textbf{Physics-Informed Kolmogorov-Arnold Networks (PIKANs)}. Unlike static sensors, this autonomous swarm actively "sniffs" the thermodynamic entropy gradient of the air to trace gas leaks back to their micro-fissure source in real-time.
        
        \item \textbf{Detailed Explanation:} \\
        The system deploys hexacopters equipped with thermal and electrochemical sensors. An onboard Edge-AI Engine (RTX 4050) uses a PIKAN architecture to calculate the \textbf{Entropy Production Rate ($\dot{S}_{gen}$)} to mathematically pinpoint the leak source (point of maximum irreversibility) with sub-millimeter precision.
        
        \item \textbf{How it addresses the problem:}
        \begin{itemize} \itemsep0cm
            \item \textbf{Speed:} Reduces detection time from $\sim$20 mins (static sensors) to $<30$ seconds (active swarm).
            \item \textbf{Precision:} Distinguishes between harmless steam and hazardous chemical leaks using thermodynamic signature analysis.
        \end{itemize}
        
        \item \textbf{Innovation:}
        \begin{itemize} \itemsep0cm
            \item First-ever PIKAN implementation on edge drones (90\% higher parameter efficiency than standard Deep Learning).
            \item \textbf{Thermodynamic Navigation:} The first system to use the Second Law of Thermodynamics as a navigation beacon.
            \item \textbf{Military-Grade Reliability:} Byzantine Fault Tolerant Swarm Consensus (PBFT) eliminates single-point failures.
        \end{itemize}
    \end{itemize}
\end{frame}

\begin{frame}{TECHNICAL APPROACH}
    \scriptsize % Compact font
    \vspace{-0.6cm}
    \begin{itemize} \itemsep0.3em
        \item \textbf{Technologies:}
        \begin{itemize}
            \item \textbf{Software:} Python (PyTorch), ROS2 (Humble), NVIDIA Omniverse (Digital Twin), \textbf{Sentinel-OS} (Drone-in-a-Box Manager).
            \item \textbf{Hardware:} Custom Hexacopter, Jetson Nano/Laptop (RTX 4050), Thermal \& MQ Sensors.
        \end{itemize}
        
        \item \textbf{Methodology (The PIKAN Engine):} \\
        The AI minimizes a Physics-Informed Loss Function ($L$) that enforces fluid dynamics laws:
        \vspace{-0.3cm}
        \[ L = \omega_{data} MSE_{obs} + \omega_{physics} || \mathcal{N}(u, C) ||^2 + \omega_{entropy} \min(\dot{S}_{gen}) \]
        \vspace{-0.4cm}
        { \tiny Term 1: Matches sensor data. Term 2 (Navier-Stokes): Physical consistency. Term 3 (Entropy Search): "Scent" signal. \par }

        \item \textbf{Entropy-Based Localization Logic:} \\
        The swarm follows the gradient of the local Entropy Production Rate ($\sigma$):
        \vspace{-0.3cm}
        \[ \sigma(t) = \int \frac{J(x,t)^2}{D \cdot p(x,t)} dx \]
    \end{itemize}
\end{frame}
